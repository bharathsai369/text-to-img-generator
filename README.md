
# Text-to-Image Generator

A React application that allows users to convert text into beautifully styled images with customizable formatting options.

## Features

- **Rich Text Formatting**: Bold, italic, underline, and text alignment options
- **Custom Styling**: 
  - Font selection from multiple font families
  - Color customization for text and background
  - Adjustable font size, line height, and letter spacing
  - Border radius and shadow effects
- **Preset Themes**: Quick theme selection for common use cases
- **Export Options**: Download as PNG or JPEG
- **Live Preview**: Real-time visualization of styling changes
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- React + TypeScript
- Vite for build tooling
- TailwindCSS for styling
- Canvas API for image generation
- Lucide React for icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

- `src/`
  - `TextToImageGenerator.tsx` - Main component for the generator
  - `components.tsx` - Reusable UI components
  - `constants.ts` - Theme and default settings
  - `utils.ts` - Canvas and helper functions
